## 🔄 **ALUR PENGGUNAAN WEBSITE**

### **📋 LANGKAH SEDERHANA:**

```
1. 🏠 Buka Website
   ↓
2. 📤 Upload Gambar Lowongan
   ↓
3. ⏳ Tunggu Proses Analisis
   ↓
4. 📊 Lihat Hasil
   ↓
5. 📥 Download Laporan (opsional)
```

### **🎯 DETAIL SETIAP LANGKAH:**

#### **Langkah 1: Buka Website**
- User masuk ke halaman utama
- Cek status sistem (harus hijau semua)
- Klik tombol "Mulai Analisis"

#### **Langkah 2: Upload Gambar**
- Drag & drop gambar ATAU klik "Pilih File"
- Gambar muncul preview
- Klik "Mulai Analisis"

#### **Langkah 3: Proses Analisis**
- Loading screen muncul
- Progress bar menunjukkan tahapan:
  - Analisis gambar dengan AI
  - Ekstrak teks dengan OCR
  - Analisis teks
  - Gabungkan hasil

#### **Langkah 4: Lihat Hasil**
- Muncul 3 hasil analisis:
  - Hasil AI dari gambar
  - Teks yang diekstrak (bisa diedit)
  - Hasil analisis teks
- Kesimpulan akhir dengan persentase

#### **Langkah 5: Aksi Selanjutnya**
- Download laporan PDF
- Analisis gambar baru
- Edit teks dan analisis ulang

## 📱 **TIPS PENGGUNAAN**

### **✅ GAMBAR YANG BAGUS:**
- Resolusi tinggi dan jelas
- Teks terbaca dengan baik
- Tidak blur atau gelap
- Format JPG/PNG/PDF

### **❌ HINDARI GAMBAR:**
- Blur atau tidak jelas
- Terlalu kecil (< 500px)
- Teks tidak terbaca
- File rusak/corrupt

### **🎯 UNTUK HASIL TERBAIK:**
- Gunakan gambar asli (bukan screenshot)
- Pastikan pencahayaan cukup
- Hindari gambar miring
- Crop jika ada bagian tidak perlu

**CekAjaYuk = Website untuk cek lowongan kerja palsu**
